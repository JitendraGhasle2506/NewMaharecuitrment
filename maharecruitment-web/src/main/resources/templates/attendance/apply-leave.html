Instructions for adding a leave record
  1. Select Leave type
  2. Select Leave category
  3. Enter the leave start and end date in yyyy-mm-dd format
  4. Enter the leave description.

Note:

a. Employee leave record added in the leave register will be represented accordingly in the employee attendance register.
